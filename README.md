# Ladle AI

## Project Overview

Ladle AI is an AI-powered recipe generation web application that helps users discover recipes based on ingredients or food preferences. Users can generate recipes using AI, upload food images for analysis, save their favorite recipes, and manage them through a secure dashboard.

The application focuses on providing a simple, responsive, and user-friendly experience while using modern web technologies.

## Features

* Secure user authentication using Supabase
* Protected routes for authenticated users
* AI recipe generation using Groq AI
* Food image analysis using Groq Vision
* Save, view, edit, and delete recipes (CRUD)
* Analytics dashboard using Recharts
* Responsive landing page
* Email support using Resend

## Tech Stack
